Taak 04 Foire Aux Questions!

Si vous avez des questions sur l’utilisation de YouTube, il y a une FAQ ci-dessous!

Comment puis-je supprimer une playlist de vidéos?

  • Dans votre liste des playlists, cliquez sur la playlist que vous souhaitez supprimer. À droite on clique sur les trois points et on sélectionne ‘Supprimer la playlist’.

Où puis-je trouver mes vidéos préférées?

  • Dans votre menu à gauche, cliquez sur la playlist “Favorites”.

Si vous faites une playlist sur un ordinateur, comment est-ce que je vois cette playlist sur une autre ordinateur?

  • Vous voyez la playlist quand vous connectez avec votre compte Google.

Comment puis-je supprimer toute une playlist?

  • Vous devez selectionner la playlist que vous voulez supprimer, à droite vous pouvez sélectionner l’option: “ Supprimer la playlist”.

Quand je sélectionne une vidéo, la vidéo ne marche pas. Comment est-ce que je la trouve maintenant?

  • Pressez sur F5 sur votre clavier.
  • Redémarrez votre ordinateur.
  • Essayez avec un autre navigateur.
  • Mettez à jour votre navigateur.

Y a-t-il une liste d’histoire ?

  • Oui, on trouve ça aussi dans le menu à gauche mais cliquez sur “Historique”.
